# Voice-to-SQL Assistant

## Overview
This project converts voice or natural language queries into SQL queries and executes them on a database.

## Features
- Speech-to-Text conversion
- Natural Language Processing
- SQL Query Generation
- Database Query Execution

## Technologies Used
- Python
- SQL
- Whisper
- Google Colab

## Example
Voice Input:
"Show all employees with salary greater than 50000"

Generated SQL:
SELECT * FROM employees WHERE salary > 50000;
